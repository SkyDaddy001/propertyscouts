Complete Flow

  1. Developer updates prices in Google Sheet (daily)
  2. n8n checks at midnight (24h cycle) for price changes
  3. If prices changed: Fetch live forex rates (USD, GBP, AED)
  4. Generate updated JSON with INR + multi-currency prices
  5. Push JSON to GitHub repository
  6. GitHub Actions rebuilds Hugo (2-3 minutes)
  7. Website updates automatically - all 21 pages reflect new prices
  8. n8n sends notification to team (prices updated)

n8n Workflow Steps

  1. Trigger: Daily at 12:00 AM IST
  2. Read Google Sheet (Developer's price data)
  3. Compare with last stored prices (detect changes)
  4. If prices changed:
    • Fetch live forex rates from API (USD, GBP, AED)
    • Calculate converted prices
    • Generate JSON file
    • Commit to GitHub /data/prices.json
    • Trigger GitHub Actions workflow
  5. GitHub Actions rebuilds Hugo (2-3 minutes)
  6. Website refreshes automatically (all 21 pages)
  7. Send Slack/Email notification to Home Realtors team:
    • "Prices updated at 12:00 AM"
    • Show which properties changed
    • Show old vs new prices
  8. If no changes detected: Skip rebuild, save resources

✅ Benefits of This System

  • Single Source of Truth: One Google Sheet, all 21 pages auto-update
  • Developer-Friendly: Just edit Google Sheet, no technical skills needed
  • Daily Automation: Updates happen at midnight automatically
  • Live Forex: USD/GBP/AED prices always accurate
  • Transparent Pricing: Min-max range shows real options
  • Urgency Messaging: "Limited time pricing" creates conversion pressure
  • No Manual Work: Zero manual page updates needed
  • Version Control: All price changes tracked in Git history
  • Fast Rebuild: Only rebuilds if prices actually changed (saves GitHub resources)
