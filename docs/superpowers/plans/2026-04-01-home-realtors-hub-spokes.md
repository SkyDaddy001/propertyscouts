# Home Realtors Hub & Spokes Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build multi-landing-page real estate marketing website with Hugo + GitHub Pages, 21 pages across 3 properties (7 personas × 3 properties), automated price management via Google Sheets + n8n + live forex conversion.

**Architecture:** 
- Hugo static site generator (fast, SEO-optimized)
- GitHub Pages hosting (free, auto-deploy)
- Google Sheets (single price source)
- n8n automation (daily price sync + forex conversion)
- GitHub Actions (CI/CD rebuild triggers)

**Tech Stack:** Hugo, GitHub Pages, GitHub Actions, n8n, Google Sheets API, OpenExchangeRates API, Netlify (fallback)

```html
    <link rel="stylesheet" href="/css/main.css">
    {{ if .Params.canonical }}<link rel="canonical" href="{{ .Params.canonical }}">{{ end }}
</head>
<body>
    {{ partial "header.html" . }}
    <main class="container">
        {{ block "content" . }}{{ end }}
    </main>
    {{ partial "footer.html" . }}
    <script src="/js/app.js"></script>
</body>
</html>
```

- [ ] **Step 2: Create single page layout**

```html
{{ define "content" }}
<article class="page">
    <h1>{{ .Title }}</h1>
    <p class="subtitle">{{ .Params.subtitle }}</p>
    
    {{ .Content }}
    
    {{ if .Params.showPrices }}
        <section class="pricing-section">
            <h2>Pricing</h2>
            <div class="price-grid">
                {{ range .Params.units }}
                    {{ partial "price-card.html" (dict "unit" . "property" $.Params.propertyId) }}
                {{ end }}
            </div>
        </section>
    {{ end }}
    
    {{ if .Params.showForm }}
        {{ partial "nurture-form.html" . }}
    {{ end }}
</article>
{{ end }}
```

- [ ] **Step 3: Create price-card partial**

```html
{{ $property := $.property }}
{{ $units := where site.Data.prices.prices "property" $property }}
{{ $unitType := .unit }}

{{ $price := index (where $units "unitType" $unitType) 0 }}

{{ if $price }}
<div class="price-card">
    <h3>{{ upper .unit }} Starting from</h3>
    <div class="price-display">
        <p class="inr">₹{{ div $price.minPrice.inr 100000 }}L - ₹{{ div $price.maxPrice.inr 100000 }}L</p>
        <p class="size">{{ $price.minSize }} - {{ $price.maxSize }} sqft</p>
    </div>
    <div class="multi-currency" id="currency-{{ $unitType }}">
        <p class="usd" data-min="{{ $price.minPrice.usd }}" data-max="{{ $price.maxPrice.usd }}">
            ${{ $price.minPrice.usd }} - ${{ $price.maxPrice.usd }} USD
        </p>
    </div>
    <p class="updated">Last updated: {{ $price.lastUpdated | dateFormat "Jan 02, 2006" }}</p>
</div>
{{ end }}
```
